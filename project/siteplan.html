TreeLand Website Plan

Planning document for my community development project

1. Site Name

TreeLand

TreeLand is the name of my project because it represents growth, opportunity, and community development through income-generating activities.

2. Site Purpose

The purpose of TreeLand is to present community development projects, attract investors, and provide information about income-generating activities that help create jobs and improve local communities.

3. Scenarios

  • How can I invest in TreeLand projects?
  • What types of activities does TreeLand offer to create jobs?

4. Color Scheme

#2E7D32
#A5D6A7
  • #2E7D32: Used for headings, header, and buttons.
  • #A5D6A7: Used for backgrounds and highlights.

5. Typography

  • Roboto: Used for body text.
  • Poppins: Used for headings.
